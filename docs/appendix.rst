Glossary
--------

.. glossary::

   Exporter
      A server application for exposing metrics to Prometheus

   IBM Z
      IBM's mainframe product line

   Prometheus
      A server application for monitoring and alerting

   Z HMC
      Hardware Management Console for IBM Z
